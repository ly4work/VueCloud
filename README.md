# VueCloud

> Vue.js 单页应用

## 技术点
- vue
- vue-router
- vuex
- axios
- webpack
- express
- sass

### 第1期 2017-05-11
- 接入one接口，实现上下翻页功能
请求由express代理，one数据每日更新。

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```
